# Number Guessing Game

Простая консольная игра "Угадай число", написанная на Python.

## Описание
Программа генерирует случайное число от 1 до 100, а пользователь пытается угадать его за ограниченное количество попыток.

## Функции
- Генерация случайного числа
- Подсказки ("больше"/"меньше")
- Ограничение попыток
- Статистика игр
- Возможность повторной игры

## Запуск
```bash
python number_guessing_game.py
